HTML


O HTML, ou Linguagem de Marcação de Hipertexto, é uma linguagem de desenvolvimento web que tem várias características, entre elas: Tags São marcadores que indicam a função de cada elemento na página, como títulos, parágrafos, imagens e links. As tags podem ser organizadas em uma estrutura hierárquica, permitindo a criação de seções e subseções. Estruturação O HTML é responsável por mostrar ao navegador como um conteúdo deve estar estruturado. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Hiperligações O elemento em HTML, com o atributo href, cria-se um hiperligação nas páginas web. Metadados O HTML cuida dos metadados para outras ferramentas. Tipos de tags As tags podem ser categorizadas em dois tipos: em “nível de bloco” (block-level) e “em linha” (inline). O HTML é o bloco de construção mais básico da web e define o significado e a estrutura do conteúdo da web. Outras tecnologias, como o CSS e o JavaScript, são usadas para descrever a aparência e a funcionalidade de uma página da web, respectivamente

CSS


O CSS, ou Cascading Style Sheets, é uma linguagem de estilização de páginas web que possui várias características, entre elas: Separação entre conteúdo e apresentação O CSS permite separar o conteúdo HTML da apresentação visual da página, facilitando a manutenção e atualização do site. Personalização Com o CSS, é possível ter vários estilos em uma página HTML, tornando as possibilidades de personalização quase infinitas. Economiza tempo e código O CSS permite estilizar tudo em um arquivo diferente, criando assim o estilo separadamente. Ajusta para diferentes telas O CSS permite ajustar o site a diferentes telas de dispositivos, exibindo corretamente o conteúdo. Adiciona efeitos É possível adicionar efeitos como sombras, bordas arredondadas e animações. Altera a aparência de elementos O CSS permite alterar a aparência de elementos na página, como cores, tamanhos, fontes e posição.

JAVASCRIPT


JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web, toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática mostrando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animados, etc. O JavaScript é uma linguagem de programação com várias características, incluindo:Alto nível. Os códigos em JavaScript são mais próximos da linguagem humana, o que facilita a interpretação. Tem suporte universal a maioria dos navegadores modernos, como Firefox, Chrome, Edge e Safari, sem a necessidade de instalar plug-ins. O JavaScript é a linguagem mais utilizada para programação de aplicações web.
